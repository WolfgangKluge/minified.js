.each()

Invokes the given function once for each item in the list.

Syntax

each(callback)

Parameters

callback
The callback function(item, index) to invoke for each list element.
item
The current list element.
index
The second the zero-based index of the current element.
The callback's return value will be ignored.
(return value)
the list

Description

Invokes the given function once for each item in the list. The function will be called with the item as first parameter and the zero-based index as second.

Example

This goes through all h2 elements of the class 'section' and changes their content: 

$('h2.section').each(function(item, index) {
    item.innerHTML = 'Section ' + index + ': ' + item.innerHTML;
});
